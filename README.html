Instrucciones para Desplegar la Aplicación Web con Docker

Sigue los pasos a continuación para desplegar la aplicación web usando Docker:

Requisitos Previos

  • Asegúrate de tener Docker instalado en tu sistema.
  • Asegúrate de tener Docker Compose instalado en tu sistema.

Pasos para Ejecutar el Contenedor

1. Verificar que la Carpeta Está Descargada

Asegúrate de que la carpeta del proyecto ya esté descargada en tu sistema. No es necesario clonar o descargar el repositorio nuevamente.

2. Ejecutar el Archivo run.bat

En la raíz de la carpeta del proyecto, encontrarás un archivo llamado run.bat. Este archivo ejecutará automáticamente el comando docker-compose up -d --build, que levantará el contenedor de Docker con tu aplicación web. Para ejecutarlo, sigue estos pasos:

./run.bat

Este archivo ejecutará los siguientes pasos automáticamente:

  • Compilar las imágenes de Docker necesarias.
  • Levantar los servicios en segundo plano (-d).

3. Acceder a la Aplicación Web

Una vez que el contenedor esté en ejecución, abre tu navegador y navega a http://localhost para ver la aplicación web en funcionamiento. La aplicación estará disponible en el puerto 80.

Detener y Eliminar el Contenedor

1. Detener el Contenedor

Para detener el contenedor en ejecución, puedes usar el siguiente comando dentro de la carpeta del proyecto:

docker-compose down

Notas Adicionales

  • El archivo run.bat se encarga de ejecutar automáticamente el comando docker-compose up -d --build.
  • Si realizas cambios en el código fuente de la aplicación, ejecuta nuevamente el archivo run.bat para reconstruir la imagen y desplegar los cambios.
  • Asegúrate de que el puerto 80 no esté siendo utilizado por otro proceso en tu máquina local.
